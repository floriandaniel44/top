Place static site images here for social sharing and favicons.

Files to add:
- opengraph.png       (recommended size: 1200x630)
- favicon.ico         (multi-size ico or 32x32)
- favicon-32x32.png   (32x32 PNG)
- favicon-16x16.png   (16x16 PNG)
- apple-touch-icon.png (180x180 PNG)

How to use:
- Add your images to this `public/` folder. Vite will copy them to the `dist` when building.
- The HTML header already references these paths (e.g. `/opengraph.png`).

If you need a quick placeholder, you can generate images with any image editor or use an online tool to export the required sizes.
